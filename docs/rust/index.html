UMSH Rust API Docs

Workspace API documentation for the UMSH Rust crates.

Protocol Book Umbrella Crate

Crates

  • umsh — umbrella crate that re-exports the workspace surface
  • umsh_core — core packet types and protocol primitives
  • umsh_crypto — cryptographic constructions and identities
  • umsh_hal — hardware abstraction traits
  • umsh_uri — UMSH URI parsing and formatting
  • umsh_chat_room — chat room payload types
  • umsh_text — text messaging payloads and wrappers
  • umsh_mac — MAC layer implementation
  • umsh_node — application-facing node layer

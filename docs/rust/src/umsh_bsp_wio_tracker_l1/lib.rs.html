3//! Board support for the Seeed Wio Tracker L1 / L1 Pro (nRF52840).
4//!
5//! Composes [`umsh-bsp-nrf52840`](../umsh_bsp_nrf52840/index.html) with
6//! the Wio Tracker L1 family's pinout and on-board peripherals.
7//!
8//! The same crate covers the L1, L1 Pro, and L1 Lite variants, which
9//! share a pin map. The L1 e-ink variant has a different display path
10//! (SPI1) and would warrant a separate BSP crate.
11//!
12//! Initial scope is *bringup only* (see
13//! `docs/firmware-plan-wio-tracker-l1.md`):
14//!
15//! - USB-CDC handles (over the nRF native USB peripheral)
16//! - user LED (P1.01, active-high) for heartbeat
17//! - SH1106 OLED I²C bus (Phase 2)
18//! - SX1262 LoRa radio + external RXEN pin (Phase 3)
19//!
20//! Future expansion (GNSS, joystick, Grove I²C, QSPI flash, buzzer,
21//! battery measurement) is welcome but is not yet implemented.
22//!
23//! See `docs/seeed-wio-tracker-l1-pro-hardware.md` for the
24//! firmware-derived hardware reference.

Struct LocalNode

pub struct LocalNode<M>
where
    M: MacBackend,
{ /* private fields */ }
Expand description

Per-identity application handle.

LocalNode owns its channel membership set via shared interior state. Different LocalNode instances (for different identities) may join different channel sets.

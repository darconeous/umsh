Expand description

A cancel-safe, frame-oriented companion transport.

Source

fn poll_recv_frame( &mut self, cx: &mut Context<'_>, ) -> Poll<Result<Vec<u8>, CompanionRadioError>>

Poll for the next complete companion frame.

Implementations keep all partial state in self, so cancellation cannot discard a partial frame or unread bytes following a completed frame.

Expand description

A context through which UMSH frames can be sent.

Both LocalNode and BoundChannel implement this trait, allowing generic code over the transport context.

Important: Transport is a context abstraction, not a security abstraction. LocalNode::send() produces a unicast frame (destination- encrypted, only the recipient can decrypt). BoundChannel::send() produces a blind unicast frame (channel-encrypted — any node with the channel key can decrypt). Generic code over Transport must not assume identical delivery or privacy properties.

Option keys carried in an MacCommand::IdentityRequest payload.

Keys follow the CoAP convention: an odd key (least-significant bit set) is critical, so a responder that does not understand it MUST NOT respond. All currently defined keys are critical. NONCE is a correlation identifier rather than a filter and does not participate in filter matching.
